<h1 align="center">📋 Task Manager</h1>

<h3 align="center">
Modern Task Management Web Application built with Laravel & Tailwind CSS
</h3>

<p align="center">
Task Manager is a responsive web application that helps users create, organize, update, 
and track their daily tasks through a clean dashboard interface.
It is designed to improve productivity with a modern UI and simple workflow.
</p>

---

### 🚀 Features
- ✅ User Authentication (Login / Register / Logout)
- ✅ Create New Tasks
- ✅ Edit Tasks
- ✅ Delete Tasks
- ✅ Task Status Management
- ✅ Search & Filter Tasks
- ✅ Responsive Sidebar Dashboard
- ✅ Success & Error Alerts
- ✅ Clean UI / UX
- ✅ Mobile Friendly Design

### ⚙️ Installation

```bash
git clone https://github.com/your-username/task-manager.git
cd task-manager
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
npm run dev
