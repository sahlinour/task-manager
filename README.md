# 📋 Task Manager

A modern **Task Management Web Application** built with **Laravel**, **Blade**, **Tailwind CSS**, and **MySQL**.  
This project helps users create, manage, update, and organize daily tasks through a clean and responsive dashboard interface.

---

## 🚀 Features

- ✅ User Authentication (Login / Register / Logout)
- ✅ Create New Tasks
- ✅ Edit Tasks
- ✅ Delete Tasks
- ✅ Task Status Management
  - To Do
  - In Progress
  - Completed
- ✅ Search & Filter Tasks
- ✅ Responsive Sidebar Dashboard
- ✅ Success & Error Alerts
- ✅ Clean UI with Tailwind CSS
- ✅ Responsive Design

---

## 🛠️ Tech Stack

- **Backend:** Laravel
- **Frontend:** Blade
- **Styling:** Tailwind CSS, Javascript
- **Database:** MySQL
- **Authentication:** Laravel Auth
